# Icon Group

將數個 icon 透過 grid 包裝

## 使用

```
{{</* icon-group */>}}

{{</* /icon-group */>}}
```

## 範例

```html
{{</* icon-group */>}}
{{</* svg-wrapper name="github" /*/>}}
{{</* svg-wrapper name="rss" /*/>}}
{{</* svg-wrapper name="twitter" /*/>}}
{{</* /icon-group */>}}
```

{{< icon-group >}}
{{< svg-wrapper name="github" />}}
{{< svg-wrapper name="rss" />}}
{{< svg-wrapper name="twitter" />}}
{{< /icon-group >}}
