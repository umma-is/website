# Terms Cloud

渲染指定的 terms

## 使用

```
{{</* terms-cloud terms="" /*/>}}
```

參數:

* `terms`: terms 的名字

## 範例

```html
{{</* terms-cloud terms="tags" */>}}
```

{{< terms-cloud terms="tags" >}}
